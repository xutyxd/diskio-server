# Diskio Client
An API client for [diskio-server](https://github.com/xutyxd/diskio-server) exposed with OpenAPI.



## 📦 Packages

### 📦 CommonJS

```js
const { DiskioAPIClient } = require('diskio-api');
```

### 📦 ESM

```js
import { DiskioAPIClient } from 'diskio-api';
```

### 📦 TypeScript

```ts
import { DiskioAPIClient } from 'diskio-api';
```
